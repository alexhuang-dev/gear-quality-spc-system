# v1.0.0

## Overview

This is the first production-style release of the Gear Quality SPC System.

It packages the project as a deployable industrial quality analysis stack with:

- deterministic SPC computation
- LangGraph orchestration
- FastAPI backend services
- SQLite-backed historical memory
- Western Electric rule evaluation
- machine-checkable harness validation
- HTML / JSON / SVG reporting
- optional PDF generation
- Streamlit dashboard
- optional Langflow visual entry

## Highlights

### Pure Python production backend

The core runtime does not depend on Langflow. Langflow is included only as an optional showcase and presentation layer.

### Deterministic quality computation

Critical values such as Cp, Cpk, control limits, historical deltas, and quality status are computed by code instead of generated by LLMs.

### Historical memory and comparison

The system persists run history in SQLite and compares current output against previous runs.

### Harness validation

The final structured output is machine-checked for internal consistency, and golden-case regression tests are included.

### Production-facing outputs

The system produces HTML reports, JSON reports, SVG charts, alert payloads, and dashboard-ready state.

## Included entry points

- FastAPI backend
- Streamlit dashboard
- Auto-runner for incoming CSV files
- Langflow final flow
- Langflow custom component

## Notes

- This release is production-oriented at the software level
- Real factory deployment still requires real process specs, real alert channels, and real MES/ERP/PLC integration decisions
